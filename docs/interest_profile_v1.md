# PKIA Interest Profile v1

---

## 1. Purpose

本文件用于定义 PKIA 用户兴趣画像。

目标：

帮助系统回答：

1. 我以前关注什么？
2. 最近出现了什么新东西？
3. 和我的方向有什么关系？
4. 是否值得投入时间？

PKIA 的核心不是发现所有信息。

而是过滤信息。

## 3. Interest Tiers

### S Tier (Core Focus)

说明：

最重要方向。

未来推荐系统必须优先保留。

包含：

- Agent
- Agent Application
- Agent Framework
- Agent Memory
- Multi-Agent
- AI Engineering

原因：

与职业目标直接相关。

任何该领域的重要项目都应优先推荐。

---

### A Tier (Strong Related)

说明：

强相关方向。

作为能力补充长期关注。

包含：

- MCP
- RAG
- AI Infrastructure
- LLMOps
- Frontier AI Trends

原因：

这些方向支撑 Agent 系统落地。

---

### B Tier (Research Background)

说明：

保留关注。

但不作为主要投入方向。

包含：

- Knowledge Graph
- Knowledge Graph Embedding
- Distributed Training
- Graph Partitioning

原因：

来源于研究背景。

未来仅在与 Agent 结合时提高优先级。

例如：

- Graph RAG
- Agent Memory + KG
- Knowledge-Augmented Agent

## 5. Recommendation Philosophy

说明 PKIA 的推荐原则。

**原则 1：** 职业价值 > 热度

**原则 2：** 长期价值 > 短期流量

**原则 3：** Agent 相关优先

**原则 4：** 可落地优先于纯概念

**原则 5：** 帮助用户减少信息焦虑

强调：

PKIA 的目标不是发现更多内容。

而是帮助用户知道：

哪些值得看。

哪些可以放心跳过。

Interest Profile 允许未来升级：

- **v2：** 增加权重系统

- **v4：** 增加长期兴趣衰减
